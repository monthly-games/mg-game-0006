## 🎮 플레이 시나리오

### 게임 시작
1. 메인 메뉴에서 "START BATTLE" 선택
2. 전투 시작: 플레이어 HP 50, 적 HP 30
3. 초기 마나 3, 초기 5장 드로우

### 전투 진행
1. **플레이어 턴**
   - 손패에서 카드 선택 (마나 부족 시 회색 표시)
   - 카드 플레이 → 효과 즉시 적용
   - "END TURN" 클릭

2. **적 턴**
   - 적이 자동 공격 (5-8 데미지)
   - 플레이어 방어력 먼저 차감
   - 자동으로 다음 턴 시작

3. **다음 턴**
   - 마나 +1 (최대 10)
   - 방어력 초기화
   - 손패 버리고 새로 5장 드로우

### 게임 종료
1. **승리**: 적 HP 0 → "VICTORY!" → RESTART
2. **패배**: 플레이어 HP 0 → "DEFEAT!" → RESTART

## 🎯 핵심 시스템

### 마나 시스템
- 시작: 3 마나
- 매 턴: +1 마나
- 최대: 10 마나
- 카드 플레이 시 소비

### 방어력 시스템
```dart
데미지 계산 = max(0, 적 공격력 - 플레이어 방어력)
방어력 = max(0, 방어력 - 적 공격력)
```

### 덱 순환
```
Deck (10장) → Hand (5장) → Discard Pile
                ↓
         덱 비면 섞어서 재활용
```
