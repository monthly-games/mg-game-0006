## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 영웅 수집/강화                                       │
│         ↓                                               │
│  2. 덱 편성/전략 설정                                    │
│         ↓                                               │
│  3. PVP 매치/리그 참여                                   │
│         ↓                                               │
│  4. 시즌 보상 수령 및 리셋 ─────────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **영웅 수집/강화**: 다양한 영웅 획득 및 능력치 강화
2. **덱 편성**: 전략적 덱 구성 및 진형 설정
3. **PVP 매치**: 다른 플레이어와의 실시간/비동기 대전
4. **시즌 보상**: 랭킹에 따른 보상 수령, 시즌 리셋

---

## 3. 재미 축 (Fun Pillars)

### 3.1 경쟁/랭킹 (Competition)
- **목표**: 다른 플레이어와의 경쟁에서 오는 긴장감과 성취감
- **핵심 경험**: 랭킹이 올라갈 때의 뿌듯함
- **주요 요소**:
  - 실시간 랭킹 시스템
  - 리그/디비전 구조
  - 시즌 보상
  - 명예의 전당

### 3.2 빌드/메타 읽기 (Meta)
- **목표**: 현재 메타를 분석하고 대응하는 전략적 재미
- **핵심 경험**: 메타를 읽고 카운터로 승리했을 때의 쾌감
- **주요 요소**:
  - 메타 분석 도구
  - 카운터 빌드 시스템
  - 리플레이 분석
  - 커뮤니티 빌드 공유

### 3.3 수집/강화 (Growth)
- **목표**: 영웅 수집과 강화를 통한 성장감
- **핵심 경험**: 새로운 영웅을 얻었을 때의 기대감
- **주요 요소**:
  - 영웅 뽑기 시스템
  - 강화/승급
  - 도감 보상
  - 스킨/외형

## 4. 디자인 레버 (Design Levers)

### 4.1 메타 순환 설계
| 레버 | 설명 |
|------|------|
| 카운터 메타 유도 | 특정 메타가 뜨면 자연스럽게 카운터 메타가 생기도록 스킬/역할 설계 |
| 구현 방향 | 가위바위보 상성, 버프/디버프 상쇄 시스템, 진형별 강약점 |

### 4.2 덱 구성 제한
| 레버 | 설명 |
|------|------|
| 덱 구성 퍼즐화 | 코스트/속성/직업 제한으로 "그냥 좋은 거 다 넣기"를 막음 |
| 구현 방향 | 총 코스트 30 제한, 같은 직업 최대 2명, 속성 밸런스 보너스 |

### 4.3 리그 긴장감
| 레버 | 설명 |
|------|------|
| 승급/강등 시스템 | 승급/강등이 모두 있는 리그 설계로 긴장 유지 |
| 구현 방향 | 3연승 → 승급전, 3연패 → 강등전, 시즌 말 보호 기간 |

### 4.4 메타 학습 지원
| 레버 | 설명 |
|------|------|
| 리플레이/덱 공유 | 상위권 리플레이/덱 공유 기능으로 메타 학습 지원 |
| 구현 방향 | 톱10 덱 공개, 리플레이 다운로드, 인기 빌드 랭킹 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 PVP 매치 수 | 10회 이상 | 매치 로그 |
| 덱 변경 빈도 | 주 5회 이상 | 덱 편집 이벤트 |
| 리그 참여율 | 80% 이상 | 시즌 중 매치 기록 유저 비율 |
| 상위권 도달률 | 20% 이상 | 골드 리그 이상 비율 |
| D7 리텐션 | 30% 이상 | 7일차 잔존율 |
