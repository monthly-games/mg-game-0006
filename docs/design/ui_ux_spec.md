# MG-0006: Hero Auto Battle UI/UX 스펙

> **게임명**: Hero Auto Battle (영웅 자동 전투)
> **장르**: 방치형 RPG
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 자동 전투 감상 최적화
- 팀 편성 UI 세로 배치
- 방치 시 세로 화면 유지

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 스테이지, 자원 |
| 하단 | 34-48dp | 네비게이션, 퀵 메뉴 |
| 좌우 | 8dp | 전투 영역 여백 |

## 3. HUD 레이아웃

```
┌─────────────────────────────────────┐
│  [스테이지 12-5]  [🪙][💎][⚡]  [⚙️]│  ← 상단 HUD
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │         적 진영              │    │
│  │    👾  👾  👾  👾  👾        │    │
│  │                              │    │
│  │         ═══════              │    │
│  │                              │    │
│  │    🗡️  🏹  🔮  🛡️  ⚔️        │    │  ← 자동 전투 영역
│  │         아군 진영            │    │
│  └─────────────────────────────┘    │
│                                     │
│  [배속: x2]  [스킵] [자동: ON]      │  ← 전투 컨트롤
├─────────────────────────────────────┤
│  [오프라인 보상: 🪙12,345 받기]      │  ← 방치 보상 배너
├─────────────────────────────────────┤
│  [전투][영웅][장비][상점][이벤트]    │  ← 하단 네비게이션
└─────────────────────────────────────┘
```

### 3.1 상단 HUD

| 요소 | 크기 | 설명 |
|------|------|------|
| 스테이지 표시 | 120×32dp | 현재 스테이지 |
| 자원 표시 | 3×(64×32dp) | 골드, 젬, 에너지 |
| 설정 버튼 | 44×44dp | 설정 |

### 3.2 전투 영역

| 요소 | 크기 | 설명 |
|------|------|------|
| 캐릭터 | 64×64dp | 아군/적 유닛 |
| HP 바 | 캐릭터 상단 | 체력 표시 |
| 스킬 이펙트 | 가변 | 자동 발동 |

### 3.3 전투 컨트롤

| 요소 | 크기 | 설명 |
|------|------|------|
| 배속 버튼 | 80×40dp | x1/x2/x3 |
| 스킵 버튼 | 80×40dp | 전투 스킵 |
| 자동 토글 | 100×40dp | ON/OFF |
