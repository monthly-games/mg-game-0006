## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 히어로 풀/리그 구조 안정화 |
| 7~10주 | 매칭 로직/전투 리플레이 UX 개선 |
| 11~12주 | 첫 정규 시즌 준비 |

**핵심 과제:**
- 매칭 시스템 안정화
- 히어로 밸런스 초기 조정
- 리플레이 시스템 완성

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 정규 시즌 1 시작 (6주 단위) |
| 5개월 | 시즌 테마 버프/제한 룰 도입 |
| 6개월 | 정규 시즌 2 + 신규 히어로 |
| 7개월 | 시즌 테마 2 + 밸런스 패치 |
| 8개월 | 정규 시즌 3 + 여름 토너먼트 |

**핵심 과제:**
- 6~8주 단위 시즌 사이클 정착
- 시즌 테마 버프 시스템 검증
- 히어로 풀 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 0013(리그 심화) 연계 설계 시작 |
| 10개월 | 상위 리그 전용 이벤트 테스트 |
| 11개월 | 토너먼트 시스템 베타 |
| 12개월 | 1주년 챔피언십 + 연말 이벤트 |

**핵심 과제:**
- 상위 리그 전용 콘텐츠
- 토너먼트 시스템 검증
- 0013 연계 준비
